# EduMyles

EduMyles is a multi-tenant school operations platform. This repository now includes a Next.js 15 App Router scaffold plus strategy documentation for Week 1 execution.

## Getting started
1. Install dependencies: `npm install`
2. Run the dev server: `npm run dev`
3. Open http://localhost:3000

## Structure
- `src/app`: App Router pages and layouts.
- `src/lib`: Shared utilities.
- `docs/`: Week 1 plan, decision log, and environment variable checklist.

## Docs
- `docs/week1-plan.md`: Day-by-day execution plan with checkpoints.
- `docs/decisions.md`: Architecture decision-log template.
- `docs/env-plan.md`: Environment variable plan and descriptions.
