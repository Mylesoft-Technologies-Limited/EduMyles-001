# EduMyles

This repository captures the Week 1 build plan for the EduMyles multi-tenant platform. It focuses on **strategy, sequencing, and decision-making** so implementation stays consistent and secure.

## Contents
- `docs/week1-plan.md`: Day-by-day execution plan with checkpoints.
- `docs/decisions.md`: Key architectural decisions and rationale.
- `docs/env-plan.md`: Environment variables to define and how they are used.

## How to use
Start with `docs/week1-plan.md` to drive the build. Log decisions and changes in `docs/decisions.md` to keep the team aligned.
